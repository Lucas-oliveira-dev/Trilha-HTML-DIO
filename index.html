Tags Textuais

Criação de Listas

As listas podem ser criadas utilizando tags ol, ul e dl. Para tags ul e ol deve-se utilizar tags li para inserção do conteúdo da lista. Listas com tags ul são listas não-ordenadas, enquanto listas com tags ol são listas ordenadas. Para criação de menus de navegação são utilizadas principalmente listas não-ordenadas.
